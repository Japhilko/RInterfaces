Formate - base package

R untersttzt von Haus aus schon einige wichtige Formate:

  • CSV (Comma Separated Values): read.csv()
  • FWF (Fixed With Format): read.fwf()
  • Tab-getrennte Werte: read.delim()

Der Arbeitsspeicher

So findet man heraus, in welchem Verzeichnis man sich gerade befindet

getwd()

So kann man das Arbeitsverzeichnis ndern:

Man erzeugt ein Objekt in dem man den Pfad abspeichert:

main.path <- "C:/" # Beispiel fr Windows
main.path <- "/users/Name/" # Beispiel fr Mac
main.path <- "/home/user/" # Beispiel fr Linux

Und ndert dann den Pfad mit setwd()

setwd(main.path)

Bei Windows ist es wichtig Slashs anstelle von Backslashs zu verwenden.

Import von Excel-Daten

  • library(foreign) ist fr den Import von fremden Datenformaten ntig
  • Wenn Excel-Daten vorliegen - als .csv abspeichern
  • Dann kann read.csv() genutzt werden um die Daten einzulesen.
  • Bei Deutschen Daten kann es sein, dass man read.csv2() wegen der Komma-Separierung braucht.
library(foreign)
?read.csv

CSV Dateien einlesen

Zunchst muss das Arbeitsverzeichnis gesetzt werden, in dem sich die Daten befinden:

Dat <- read.csv("schuldaten_export.csv")

Wenn es sich um Deutsche Daten handelt:

Dat <- read.csv2("schuldaten_export.csv")
